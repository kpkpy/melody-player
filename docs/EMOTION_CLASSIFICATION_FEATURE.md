**分析维度：**
- 🎵 **元数据关键词** - 从歌曲标题、艺术家、专辑提取情绪词

## 📊 技术架构

### 分析流程
```
歌曲输入 → 元数据提取 → 特征估算 → 情绪评分 → 场景匹配 → 分类输出
```

### 核心算法
```typescript
// 1. 提取 BPM（节奏速度）
bpm = extractBPM(song.genre, song.metadata)

// 2. 关键词匹配
if (title.includes('happy', 'love', 'smile')) {
  emotionScores['happy'] += 0.2
}

// 3. BPM 情绪映射
if (bpm < 70) emotions = ['relax', 'sleep']
if (bpm > 140) emotions = ['energetic', 'workout']

// 4. 综合评分
finalScore = keywordScore * 0.4 + bpmScore * 0.3 + featureScore * 0.3
```

### 特征估算
| 特征 | 说明 | 范围 |
|------|------|------|
| BPM | 节奏速度 | 0-200 |
| Energy | 能量强度 | 0-1 |
| Danceability | 舞曲度 | 0-1 |
| Valence | 情绪价质（正面/负面） | 0-1 |
| Acousticness | 原声度 | 0-1 |

## 🎯 识别规则

### BPM 情绪映射
| BPM 范围 | 情绪类型 | 典型场景 |
|----------|----------|----------|
| 0-70 | 放松、冥想、睡眠 | 睡前、瑜伽 |
| 70-90 | 放松、浪漫、怀旧 | 晚餐、回忆 |
| 90-120 | 开心、专注、驾驶 | 工作、开车 |
| 120-140 | 开心、激昂、运动 | 派对、健身 |
| 140+ | 激昂、运动、派对 | 高强度训练 |

### 流派特征
| 流派 | BPM | Energy | Danceability | 主要情绪 |
|------|-----|--------|--------------|----------|
| Ambient | 60 | 0.2 | 0.2 | Relax |
| Classical | 80 | 0.4 | 0.3 | Relax, Focus |
| Pop | 110 | 0.7 | 0.7 | Happy |
| Rock | 120 | 0.8 | 0.5 | Energetic |
| Electronic | 128 | 0.8 | 0.9 | Party |
| Hip Hop | 95 | 0.7 | 0.8 | Energetic |
| Metal | 140 | 0.9 | 0.5 | Energetic |
| Jazz | 90 | 0.5 | 0.5 | Relax, Romantic |

## 💡 实际应用场景

### 1. 早起唤醒
```
场景：Morning
特点：BPM > 100, Energy > 0.6, Valence > 0.5
推荐：轻快流行音乐
```

### 2. 工作专注
```
场景：Work / Study
特点：Energy 0.3-0.7, Instrumentalness > 0.5
推荐：轻音乐、古典、氛围音乐
```

### 3. 运动健身
```
场景：Workout
特点：BPM > 140, Energy > 0.7
推荐：EDM、Hip Hop、Rock
```

### 4. 睡前放松
```
场景：Sleep / Meditation
特点：BPM < 80, Energy < 0.4, Acousticness > 0.6
推荐：Ambient、古典钢琴
```
