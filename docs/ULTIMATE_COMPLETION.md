## 📝 快速开始指南

### 1. 安装依赖
```bash
cd melody
npm install
```

### 2. 运行开发版
```bash
npm run dev
```

### 3. 测试功能
```
基础控制:
- 点击封面大图播放音乐
- 点击「📊 统计」查看数据
- 点击「💿」开启黑胶模式

交互增强:
- 按 Ctrl+Alt+P 播放/暂停
- 点击托盘图标恢复窗口
- 点击「📄」开启桌面歌词

窗口模式:
- 点击「📱」进入迷你模式
- 点击「🖼️」进入画中画模式

智能功能:
- 在统计页面添加情感标签
- 查看智能推荐歌曲
```

### 4. 构建正式版
```bash
npm run build
```
