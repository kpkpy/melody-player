## 🚀 使用方法

### 1. 访问统计面板
- 点击侧边栏导航的「📊 统计」
- 查看听歌数据概览

### 2. 四个标签页
- **总览**：总体数据 + 图表 + 最近播放
- **歌曲**：最常播放歌曲排行
- **艺术家**：最常播放艺术家排行
- **趋势**：每日听歌量趋势

### 3. 标注情感标签
- 在歌曲排行页面
- 点击歌曲右侧的「+ 标注」按钮
- 选择情感标签
- 保存

### 4. 获取推荐
- 系统自动基于播放历史推荐
- 同艺术家优先
- 同情感标签辅助

## 📁 数据存储

**存储位置**：
```
%APPDATA%/melody-player/listening-stats.json (Windows)
~/Library/Application Support/melody-player/listening-stats.json (macOS)
~/.config/melody-player/listening-stats.json (Linux)
```

**数据结构**：
```json
{
  "totalPlayTime": 12345,
  "totalPlays": 567,
  "songs": [
    {
      "id": "song1",

  "hourlyPlays": { "14": 25, ... },

```

## 📈 智能推荐算法

### 推荐优先级
```
1. 同艺术家（最高权重）
   - 播放次数排序
   - 取 Top 5

2. 同情感标签（中权重）
   - 共享情感标签
   - 播放次数排序
   - 取 Top 5

3. 去重合并
   - 使用 Map 去重
   - 返回 Top 10
```

### 场景歌单算法
```
专注模式:
  - 播放次数 > 5（熟悉度）
  - 完成率 > 70%（吸引力）
  - 取 Top 30

运动/放松/派对:
  - 情感标签过滤
  - 播放次数排序
  - 取 Top 30
```

## 🎯 用户体验流程

### 第一次使用
```
1. 用户打开统计页面
2. 看到「暂无统计数据」（因为还没播放）
3. 开始播放音乐
4. 系统自动记录播放行为
```

### 积累数据后
```
1. 回到统计页面
2. 看到总播放次数、时长
3. 查看最常播放歌曲排行
4. 点击「+ 标注」添加情感标签
5. 系统根据标签生成推荐
```

### 使用智能功能
```
1. 点击「专注模式」
2. 系统筛选高完成率歌曲
3. 生成专注歌单
4. 开始播放

或：
1. 在歌曲排行榜点击「+ 标注」
2. 选择「运动」标签
3. 下次打开「运动模式」
4. 自动包含这首歌
```
