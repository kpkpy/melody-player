## 🎨 核心功能亮点

### 1. **桌面歌词** 🎤
```
特点：
- 透明悬浮窗口
- 鼠标穿透（锁定模式）
- 实时歌词显示
- 右键菜单控制
- 双击锁定/解锁
```

### 2. **全局热键** ⌨️
```
支持：
```

### 3. **听歌统计** 📈
```
展示：
- 总播放次数、时长
- 每小时/每周分布图
- 每日趋势热力图
- 歌曲/艺术家排行 Top 10

价值：了解自己的音乐品味
```

### 4. **智能推荐** 🤖
```
算法：
- 同艺术家优先推荐
- 同情感标签辅助

场景：
- 专注模式：高完成率歌曲
- 运动模式：激昂/运动标签
- 放松模式：放松/悲伤标签
- 派对模式：开心/激昂标签
```

## 🎯 技术架构

### 主进程（Main Process）
```typescript
管理模块：
- hotkeyManager.ts      → 全局热键注册/监听
- trayManager.ts        → 系统托盘菜单
- desktopLyrics.ts      → 桌面歌词窗口
- statsManager.ts       → 统计数据管理
- index.ts              → 集成所有功能
```

### 渲染进程（Renderer Process）
```vue
视图组件：
- StatsView.vue         → 统计面板
- PlayerBar.vue         → 播放栏（新增桌面歌词按钮）
- Sidebar.vue           → 侧边栏（新增统计导航）

状态管理：
- player.ts             → 热键监听 + 播放统计
```

### 通信机制（IPC）
```typescript
主进程 → 渲染进程（事件）：
- global:playPause      → 热键触发播放暂停
- global:next           → 热键触发下一首
- global:previous       → 热键触发上一首

渲染进程 → 主进程（调用）：
- stats.startPlaying()  → 开始播放记录
- stats.getStats()      → 获取统计数据
- stats.addEmotionTag() → 添加情感标签
- desktopLyrics.toggle()→ 开关桌面歌词
```
